# AI Digest

Daily AI news digest generated by a scheduled Claude Code agent.

## How it works

1. A scheduled agent runs every morning at 8:00 AM CET/CEST
2. It searches the web for the latest AI news from the past 24 hours
3. It reads `preferences.md` to filter and rank results
4. It creates a digest file in `digests/` and opens a PR

## Giving feedback

Edit `preferences.md` to steer future digests:
- Add topics under **"More like this"** to see more coverage
- Add topics under **"Less like this"** to reduce coverage
